Datenschutzerklärung

Art der erhobenen Daten

Die Website greift auf die folgenden personenbezogenen Daten Ihres Spotify-Kontos zu:

  • Ihren Namen, Ihren Benutzernamen, Ihr Profilbild, Ihre Follower und Ihre öffentlichen Playlists
  • Ihre meistgehörten Künstler und Inhalte auf Spotify

Zusätzlich werden beim Abruf von Inhalten über die Spotify API technisch bedingte Daten wie Ihre IP-Adresse, Datum und Uhrzeit des Abrufs sowie der Name der abgerufenen Daten an Spotify übermittelt.

Zweck der Datenverarbeitung

Die Daten werden verwendet, um Ihnen personalisierte Inhalte auf dieser Website anzuzeigen. Die technisch bedingten Daten wie die IP-Adresse werden benötigt, um eine Verbindung mit der Spotify API herzustellen und die gewünschten Informationen abzurufen.

Rechtsgrundlage der Verarbeitung

Die Verarbeitung der personenbezogenen Daten erfolgt auf Grundlage Ihrer Einwilligung, die Sie durch das Einloggen mit Ihrem Spotify-Konto und die Verwendung der API erteilen.

Weitergabe an Dritte

Personenbezogene Daten werden nicht an Dritte weitergegeben, außer an Spotify AB, wenn Daten über die Spotify API abgerufen werden. Spotify verarbeitet diese Daten gemäß ihrer eigenen Datenschutzrichtlinie.

Sie haben das Recht, Auskunft über die von mir verarbeiteten personenbezogenen Daten zu verlangen, sowie das Recht auf Berichtigung, Sperrung oder Löschung dieser Daten. Außerdem haben Sie das Recht, Ihre Einwilligung zur Verarbeitung der Daten jederzeit zu widerrufen.

Wenn Sie die Verknüpfung zwischen Ihrem Spotify-Konto und dieser Website entfernen möchten, können Sie dies jederzeit unter https://www.spotify.com/account/apps/ tun.
